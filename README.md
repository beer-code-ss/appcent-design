# AppCent — Смотри рекламу и зарабатывай!

Этот проект — просто дизайн, который я делал когда-то давно для тренировки и экспериментов с вёрсткой.

Здесь только HTML, CSS и немного JS для интерфейса. Нет сложной логики, серверной части или базы данных.

---

## О проекте
- **Тип:** Учебный/дизайнерский макет
- **Назначение:** Ознакомление с вёрсткой и UI
- **Статус:** Не поддерживается, выложено для примера

---

## Как использовать
1. Скачайте или клонируйте репозиторий
2. Откройте `index.html` в браузере

---

## Важно
Проект не предназначен для коммерческого использования. Все материалы предоставлены только для ознакомления.
